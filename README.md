# Face Recognition
- Face Unlock using OpenCV in Python.

## How to use:-
- Place some face sample images in folder(*You have to specify the path*) using FaceSamples.py which you will use to unlock.
- Generated sample images will be used for training classifier.
- To test face unlock, run FaceUnlock.py.

## NOTE:-
- Install all the required python modules and then run.
- Use the given haar face classifier.

## Programming Language:-
- Python.
